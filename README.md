# 🍕 Vico del Carmine - Menu Database

Repository centrale per i dati strutturati del menù di **Vico del Carmine**.

## 🚀 Architettura Data-First
Questa repository non contiene solo testo, ma dati pronti per essere consumati da:
- [x] **GitHub Pages**: Il menù web navigabile dai clienti.
- [ ] **Digital Signage**: Schermi in sala (prossimamente).
- [ ] **App Ordini**: Integrazione futura.

## 🔗 Endpoint API (JSON)
I dati sono accessibili pubblicamente via:
`https://vicodelcarmine.github.io/menu/menu.json`

## 🛠️ Come aggiornare il menù
1. Modifica il file `menu.json`.
2. Fai il push delle modifiche.
3. Il frontend si aggiornerà automaticamente leggendo il nuovo file.

## 📋 Struttura Categorie
- **Specialità Gourmet**: Le nostre pizze stagionali.
- **Primi Piatti**: Pasta fresca e tradizione.
- **Secondi Piatti**: Carne e pesce alla carta.
- **Pizze**: La nostra selezione napoletana (3 pagine).
- **Insalatone & Contorni**: Per chi cerca leggerezza.
- **Birre Poretti**: Selezione alla spina e bottiglia.
- **Menù Fissi**: Proposte a prezzo fisso.
